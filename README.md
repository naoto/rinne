# Rinne

Rinne - Rinne is snake case convert to camel case.

## Installation

Add this line to your application's Gemfile:

    gem 'rinne'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rinne

## Usage

```ruby
require 'rinne'

Rinne.tensei("rine_tensei") #=> "RinneTensei"
```
